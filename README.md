Phase 01 Project Summary: UrbanVibe Fashion Store
Project Overview: UrbanVibe is a front-end fashion store application designed to provide a
premium shopping experience. This first phase focused on UI/UX design, core interface
implementation, and local state management for shopping features.

Design & Prototyping: The application prototype was originally designed using Figma. The
interactive prototype is available in the project files to demonstrate the initial design concept
and user flow.

Implemented Screens: The application includes a comprehensive set of screens covering the
full user journey:

• Onboarding: Splash Screen, Login, and Registration.

• Main Hub: Home Screen (Product Browsing (product details, product listing
(women, men, kid)), Wishlist, and User Profile.

• Checkout Flow: Cart, Checkout Details, Card Details, and Payment Success screens.
Key Features in Phase 01

• Navigation: All screens are fully linked with smooth transitions to provide a realistic
app flow.

• Authentication Logic: Implemented UI-level logic for Login and Logout
functionality.

• Wishlist Management: Users can add products to their favorites and remove them
dynamically.

• Advanced Cart System: Add/Remove products from the cart.

o Increase or decrease product quantities within the cart view.

o Dynamic calculation of total price, including shipping costs.
